接下来，我们要学习前端部分。首先，请确保你学习了第一部分，有关 TypeScript 语言基础的内容。

在开始之前，我们需要先了解什么是前后端。  
想象一下，你现在正在编写一个计算器程序。为了实现加法，你需要写一个这样的函数：
```typescript
function add(a: number, b: number): number {
  return a + b;
}
```
现在，你可以用一点简单的代码来修饰它，让它可以被用户直接使用：
```typescript
// 获取用户输入
const a = parseInt(prompt("请输入第一个数字"));
const b = parseInt(prompt("请输入第二个数字"));
// 计算结果
const result = add(a, b);
// 显示结果
alert(`结果是：${result}`);
```
这样的话，你就可以得到一个简单的计算机程序：
```
请输入第一个数字: 1
请输入第二个数字: 2
结果是：3
```
在这个例子中，`add` 函数就是 `后端`，而用户输入和输出的部分就是 `前端`。`后端` 负责处理数据，`前端` 负责和用户交互。但是通常来说，`前端` 不会这么草率的让用户使用命令行输入数据和获取数据（除了一些古老的系统），而是会使用更加友好的界面，例如：
